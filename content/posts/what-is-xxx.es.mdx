Seedance 2.0 es un marco de desarrollo SaaS de IA basado en NextJS, con componentes ricos incorporados y SDKs de IA para ayudarte a completar rápidamente el desarrollo de proyectos SaaS de IA.

## Requisitos del Entorno

Para asegurar que puedas ejecutar y desarrollar exitosamente Seedance 2.0 localmente, por favor instala el entorno necesario en tu computadora.

Aquí están las configuraciones de entorno recomendadas para sistemas Mac:

1. Usa [fnm](https://github.com/Schniz/fnm) para la gestión de versiones de NodeJS

Versiones recomendadas para NodeJS y npm:

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. Usa [pnpm](https://pnpm.io/installation) para la gestión de paquetes

Instala pnpm globalmente:

```bash
$ npm install -g pnpm
```

Versión recomendada para pnpm:

```bash
$ pnpm -v
9.15.0
```

3. Instala [git](https://git-scm.com/downloads) y configura [ssh key](https://docs.github.com/en/authentication/connecting-to-github-with-ssh) para asegurar que puedes acceder a tu repositorio de Github

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> Para otros sistemas operativos, por favor busca los métodos de configuración de entorno correspondientes. Antes de ejecutar el código de Seedance 2.0, por favor asegúrate de que nodejs y pnpm estén instalados en el entorno de tu computadora.

## Herramientas de IA

Para entender mejor el marco de Seedance 2.0 y desarrollar proyectos más eficientemente, recomendamos usar las siguientes herramientas de IA:

| Nombre                                   | Descripción            |
| ---------------------------------------- | ---------------------- |
| [Cursor](https://www.cursor.com/)        | Editor IA              |
| [Windsurf](https://codeium.com/windsurf) | Editor IA              |
| [v0](https://v0.dev/)                    | Generar Componentes UI |
| [Claude](https://claude.ai/)             | Chat IA                |
| [ThinkAny](https://thinkany.ai/)         | Búsqueda IA            |

## Plugins de Editor IA

Para formatear mejor el código y depurar interfaces, recomendamos instalar los siguientes plugins en VS Code / Cursor / Windsurf:

- [REST Client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client) Depuración de Interfaz
- [Prettier - Code formatter](https://marketplace.cursorapi.com/items?itemName=esbenp.prettier-vscode) Formateo de Código
