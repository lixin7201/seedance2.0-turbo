YourAppName é um framework de desenvolvimento SaaS de IA baseado em NextJS, com componentes ricos integrados e SDKs de IA para ajudá-lo a concluir rapidamente o desenvolvimento do projeto SaaS de IA.

## Requisitos de Ambiente

Para garantir que você possa executar e desenvolver o YourAppName localmente com sucesso, instale o ambiente necessário em seu computador.

Aqui estão as configurações de ambiente recomendadas para sistemas Mac:

1. Use [fnm](https://github.com/Schniz/fnm) para gerenciamento de versão do NodeJS

Versões recomendadas para NodeJS e npm:

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. Use [pnpm](https://pnpm.io/installation) para gerenciamento de pacotes

Instale o pnpm globalmente:

```bash
$ npm install -g pnpm
```

Versão recomendada para pnpm:

```bash
$ pnpm -v
9.15.0
```

3. Instale [git](https://git-scm.com/downloads) e configure [ssh key](https://docs.github.com/en/authentication/connecting-to-github-with-ssh) para garantir que você possa acessar seu repositório Github

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> Para outros sistemas operacionais, pesquise os métodos de configuração de ambiente correspondentes. Antes de executar o código YourAppName, certifique-se de que o nodejs e o pnpm estejam instalados no ambiente do seu computador.

Para entender melhor o framework YourAppName e desenvolver projetos de forma mais eficiente, recomendamos o uso das seguintes ferramentas de IA:
