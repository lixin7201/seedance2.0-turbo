Seedance 2.0 — это фреймворк разработки AI SaaS на основе NextJS со встроенными богатыми компонентами и AI SDK, которые помогут вам быстро завершить разработку проектов AI SaaS.

## Требования к среде

Чтобы убедиться, что вы можете успешно запускать и разрабатывать Seedance 2.0 локально, пожалуйста, установите необходимую среду на свой компьютер.

Вот рекомендуемые конфигурации среды для систем Mac:

1. Используйте [fnm](https://github.com/Schniz/fnm) для управления версиями NodeJS

Рекомендуемые версии для NodeJS и npm:

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. Используйте [pnpm](https://pnpm.io/installation) для управления пакетами

Установите pnpm глобально:

```bash
$ npm install -g pnpm
```

Рекомендуемая версия для pnpm:

```bash
$ pnpm -v
9.15.0
```

3. Установите [git](https://git-scm.com/downloads) и настройте [ssh key](https://docs.github.com/en/authentication/connecting-to-github-with-ssh), чтобы обеспечить доступ к вашему репозиторию Github

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> Для других операционных систем, пожалуйста, найдите соответствующие методы настройки среды. Перед запуском кода Seedance 2.0 убедитесь, что nodejs и pnpm установлены в вашей компьютерной среде.
