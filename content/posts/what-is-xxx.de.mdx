YourAppName ist ein KI-SaaS-Entwicklungsframework basierend auf NextJS, mit integrierten umfangreichen Komponenten und KI-SDKs, um Ihnen zu helfen, KI-SaaS-Projekte schnell abzuschließen.

## Umgebungsanforderungen

Um sicherzustellen, dass Sie YourAppName erfolgreich lokal ausführen und entwickeln können, installieren Sie bitte die erforderliche Umgebung auf Ihrem Computer.

Hier sind die empfohlenen Umgebungskonfigurationen für Mac-Systeme:

1. Verwenden Sie [fnm](https://github.com/Schniz/fnm) für die NodeJS-Versionsverwaltung

Empfohlene Versionen für NodeJS und npm:

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. Verwenden Sie [pnpm](https://pnpm.io/installation) für die Paketverwaltung

Installieren Sie pnpm global:

```bash
$ npm install -g pnpm
```

Empfohlene Version für pnpm:

```bash
$ pnpm -v
9.15.0
```

3. Installieren Sie [git](https://git-scm.com/downloads) und konfigurieren Sie [ssh-Schlüssel](https://docs.github.com/en/authentication/connecting-to-github-with-ssh), um sicherzustellen, dass Sie auf Ihr Github-Repository zugreifen können

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> Für andere Betriebssysteme suchen Sie bitte nach den entsprechenden Umgebungskonfigurationsmethoden. Bevor Sie YourAppName-Code ausführen, stellen Sie bitte sicher, dass nodejs und pnpm in Ihrer Computerumgebung installiert sind.
