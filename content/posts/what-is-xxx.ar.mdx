Seedance 2.0 هو إطار عمل لتطوير SaaS بالذكاء الاصطناعي يعتمد على NextJS، مع مكونات غنية مدمجة و SDKs للذكاء الاصطناعي لمساعدتك على إكمال تطوير مشاريع SaaS بالذكاء الاصطناعي بسرعة.

## متطلبات البيئة

لضمان قدرتك على تشغيل وتطوير Seedance 2.0 بنجاح محليًا، يرجى تثبيت البيئة اللازمة على جهاز الكمبيوتر الخاص بك.

إليك تكوينات البيئة الموصى بها لأنظمة Mac:

1. استخدم [fnm](https://github.com/Schniz/fnm) لإدارة إصدارات NodeJS

الإصدارات الموصى بها لـ NodeJS و npm:

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. استخدم [pnpm](https://pnpm.io/installation) لإدارة الحزم

قم بتثبيت pnpm عالميًا:

```bash
$ npm install -g pnpm
```

الإصدار الموصى به لـ pnpm:

```bash
$ pnpm -v
9.15.0
```

3. قم بتثبيت [git](https://git-scm.com/downloads) وتكوين [مفتاح ssh](https://docs.github.com/en/authentication/connecting-to-github-with-ssh) للتأكد من قدرتك على الوصول إلى مستودع Github الخاص بك

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> بالنسبة لأنظمة التشغيل الأخرى، يرجى البحث عن طرق تكوين البيئة المقابلة. قبل تشغيل رمز Seedance 2.0، يرجى التأكد من تثبيت nodejs و pnpm في بيئة الكمبيوتر الخاص بك.
