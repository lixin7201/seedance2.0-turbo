Seedance 2.0は、NextJSをベースにしたAI SaaS開発フレームワークであり、豊富なコンポーネントとAI SDKが組み込まれており、AI SaaSプロジェクトの開発を迅速に完了するのに役立ちます。

## 環境要件

Seedance 2.0をローカルで正常に実行および開発できるようにするには、コンピューターに必要な環境をインストールしてください。

Macシステムの推奨環境構成は次のとおりです。

1. NodeJSバージョン管理には[fnm](https://github.com/Schniz/fnm)を使用してください

NodeJSとnpmの推奨バージョン：

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. パッケージ管理には[pnpm](https://pnpm.io/installation)を使用してください

pnpmをグローバルにインストールします：

```bash
$ npm install -g pnpm
```

pnpmの推奨バージョン：

```bash
$ pnpm -v
9.15.0
```

3. [git](https://git-scm.com/downloads)をインストールし、[ssh key](https://docs.github.com/en/authentication/connecting-to-github-with-ssh)を設定して、Githubリポジトリにアクセスできるようにしてください

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> その他のオペレーティングシステムについては、対応する環境構成方法を検索してください。Seedance 2.0コードを実行する前に、nodejsとpnpmがコンピューター環境にインストールされていることを確認してください。

## AIツール

Seedance 2.0フレームワークをよりよく理解し、プロジェクトをより効率的に開発するために、次のAIツールを使用することをお勧めします。

| 名前                                       | 説明                   |
| ---------------------------------------- | ---------------------- |
| [Cursor](https://www.cursor.com/)        | AIエディタ              |
| [Windsurf](https://codeium.com/windsurf) | AIエディタ              |
| [v0](https://v0.dev/)                    | UIコンポーネントの生成   |
| [Claude](https://claude.ai/)             | AIチャット              |
| [ThinkAny](https://thinkany.ai/)         | AI検索                 |

## AIエディタプラグイン

コードをより適切にフォーマットし、インターフェースをデバッグするために、VS Code / Cursor / Windsurfに次のプラグインをインストールすることをお勧めします。

- [REST Client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client) インターフェースデバッグ
- [Prettier - Code formatter](https://marketplace.cursorapi.com/items?itemName=esbenp.prettier-vscode) コードフォーマット
