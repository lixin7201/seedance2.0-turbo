Seedance 2.0 is een AI SaaS-ontwikkelingsframework gebaseerd op NextJS, met ingebouwde rijke componenten en AI SDK's om u te helpen de ontwikkeling van AI SaaS-projecten snel te voltooien.

## Omgevingsvereisten

Om ervoor te zorgen dat u Seedance 2.0 succesvol lokaal kunt uitvoeren en ontwikkelen, installeert u de benodigde omgeving op uw computer.

Hier zijn de aanbevolen omgevingsconfiguraties voor Mac-systemen:

1. Gebruik [fnm](https://github.com/Schniz/fnm) voor NodeJS-versiebeheer

Aanbevolen versies voor NodeJS en npm:

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. Gebruik [pnpm](https://pnpm.io/installation) voor pakketbeheer

Installeer pnpm wereldwijd:

```bash
$ npm install -g pnpm
```

Aanbevolen versie voor pnpm:

```bash
$ pnpm -v
9.15.0
```

3. Installeer [git](https://git-scm.com/downloads) en configureer [ssh key](https://docs.github.com/en/authentication/connecting-to-github-with-ssh) om ervoor te zorgen dat u toegang heeft tot uw Github-repository

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> Voor andere besturingssystemen, zoek naar de bijbehorende omgevingsconfiguratiemethoden. Voordat u Seedance 2.0-code uitvoert, moet u ervoor zorgen dat nodejs en pnpm in uw computeromgeving zijn geïnstalleerd.

| Naam                                     | Beschrijving           |
