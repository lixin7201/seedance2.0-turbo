Seedance 2.0 to framework programistyczny AI SaaS oparty na NextJS, z wbudowanymi bogatymi komponentami i pakietami SDK AI, które pomogą Ci szybko ukończyć rozwój projektu AI SaaS.

## Wymagania Środowiskowe

Aby upewnić się, że możesz pomyślnie uruchamiać i rozwijać Seedance 2.0 lokalnie, zainstaluj niezbędne środowisko na swoim komputerze.

Oto zalecane konfiguracje środowiska dla systemów Mac:

1. Użyj [fnm](https://github.com/Schniz/fnm) do zarządzania wersjami NodeJS

Zalecane wersje dla NodeJS i npm:

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. Użyj [pnpm](https://pnpm.io/installation) do zarządzania pakietami

Zainstaluj pnpm globalnie:

```bash
$ npm install -g pnpm
```

Zalecana wersja dla pnpm:

```bash
$ pnpm -v
9.15.0
```

3. Zainstaluj [git](https://git-scm.com/downloads) i skonfiguruj [ssh key](https://docs.github.com/en/authentication/connecting-to-github-with-ssh), aby zapewnić dostęp do repozytorium Github

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> W przypadku innych systemów operacyjnych wyszukaj odpowiednie metody konfiguracji środowiska. Przed uruchomieniem kodu Seedance 2.0 upewnij się, że nodejs i pnpm są zainstalowane w środowisku komputera.

## Narzędzia AI

Aby lepiej zrozumieć framework Seedance 2.0 i wydajniej rozwijać projekty, zalecamy korzystanie z następujących narzędzi AI:

| Nazwa                                    | Opis                   |
| ---------------------------------------- | ---------------------- |
| [Cursor](https://www.cursor.com/)        | Edytor AI              |
| [Windsurf](https://codeium.com/windsurf) | Edytor AI              |
| [v0](https://v0.dev/)                    | Generuj Komponenty UI  |
| [Claude](https://claude.ai/)             | Czat AI                |
| [ThinkAny](https://thinkany.ai/)         | Wyszukiwanie AI        |
