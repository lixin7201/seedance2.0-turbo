YourAppName est un cadre de développement SaaS IA basé sur NextJS, avec des composants riches intégrés et des SDK IA pour vous aider à terminer rapidement le développement de projets SaaS IA.

## Exigences de l'Environnement

Pour vous assurer que vous pouvez exécuter et développer YourAppName localement avec succès, veuillez installer l'environnement nécessaire sur votre ordinateur.

Voici les configurations d'environnement recommandées pour les systèmes Mac :

1. Utilisez [fnm](https://github.com/Schniz/fnm) pour la gestion de version NodeJS

Versions recommandées pour NodeJS et npm :

```bash
$ node -v
v22.2.0

$ npm -v
10.7.0
```

2. Utilisez [pnpm](https://pnpm.io/installation) pour la gestion des paquets

Installez pnpm globalement :

```bash
$ npm install -g pnpm
```

Version recommandée pour pnpm :

```bash
$ pnpm -v
9.15.0
```

3. Installez [git](https://git-scm.com/downloads) et configurez [clé ssh](https://docs.github.com/en/authentication/connecting-to-github-with-ssh) pour vous assurer que vous pouvez accéder à votre dépôt Github

```bash
$ git --version
git version 2.39.3 (Apple Git-146)
```

> Pour les autres systèmes d'exploitation, veuillez rechercher les méthodes de configuration d'environnement correspondantes. Avant d'exécuter le code YourAppName, veuillez vous assurer que nodejs et pnpm sont installés dans l'environnement de votre ordinateur.
